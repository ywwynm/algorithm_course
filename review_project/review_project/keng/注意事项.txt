1. 边界问题：遍历或访问时，边界如何确定？小心心里想的坐标是x、y，结果赋值成了i、j（一般代表行和列，也就是y、x），导致边界设置错误（题1B）

2. 建图时，MAX_NODE和MAX_EDGE可以设置得大一些，有些无良的题目喜欢给一个不对的上界，按照那个设置没准就RuntimeError了

3. 最好少用float，应该都可以用double代替

4. 输入的数据不要用float保存，可能会导致莫名其妙的TLE（当然一般来说如果输入都是int的话，基本不会这么做，这个提醒只是防止脑抽）

5. float、double比较的时候小心一些，如判断相等的时候，可能==是不足够的

6. 还是边界问题，一定要想清楚循环处理的终值……

7. 有时候结果可以用short表示的话，可以适当地用short代替int=。=降低内存开销

8. 遇到TLE的时候，想一想哪些地方可以挤牙膏（ME）

9. bool 数组默认值为true，最好写成bool b[x] = { false }

10. MG: 读好题目，充分理解题意...

11. 记得每组输入都要对某些全局变量进行初始化
